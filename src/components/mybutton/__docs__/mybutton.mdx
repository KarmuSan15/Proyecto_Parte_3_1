# MyButton
Este componente es un botón con personalizaciones en color, tamaño, bordes, etc.

#### Example - Default
<Canvas of={MyButtonStories.Default} /> {/* Visualización del estado por defecto */}

## Uso
```tsx
import { MyButton } from 'milibreria';

function Example() {
  return (
    <MyButton
      text="Hola"
      txtcolor="white"
      bgcolor="blue"
      borderColor="black"
      borderWidth="1px"
      size="medium"
      hoverTxtColor="yellow"
      hoverBgColor="green"
      disabled={false}
      onClick={() => alert("¡Hola!")}
    />
  );
}
